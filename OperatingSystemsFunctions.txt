Controlling the Overall Performance - One main function of an operating system is to maintain proper performance. To do this the device mointors service requests and the systems responses. This helps improve the overall performance of the device and eliminate any disruptions that alter the performance rate.
Detecting and Handling Errors - operating systems have the ability to detect and eliminate errors. In order to do so these operating systems provide maximum security to prevent any type of breach from occurring. Any bugs or errors that occur while performing a task are detected by the operating system.
